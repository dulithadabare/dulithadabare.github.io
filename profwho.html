ProfWho - iOS App and Web Portal

A native iOS mobile application, a website and a Angular based web application for identifying and retrieving information about faculty members in the School of Computer Science and Engineering (SCSE) at Nanyang Technology University in Singapore.

The system was developed during an internship under Dr. Owen Noel Newton Fernando of SCSE, following design guidelines developed for the UI and UX of the iOS application, website and web application.

The iOS app integrated Vuforia Image Recognition API to identify unique objects using the phone camera and uses Estimote Bluetooth API to identify unique objects using Estimote Low-Energy Bluetooth Beacons.

The web application allows users to to add information, assign user profiles to bluetooth beacons and upload profile images to be used for image recognition.
